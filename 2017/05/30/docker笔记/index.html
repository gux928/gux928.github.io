ghost docker安装记录

2017-05-30

从来只是磨刀，至于砍柴……将来再说吧

工具：

  • VPS：ubuntu 14.04@SJC LIZCAT
  • Docker
  • Docker-compose
  • Ghost-Blog

过程：

  1. 安装docker及docker－compose：
    $ wget -qO- https://get.docker.com/ | sh比较好用；pip install -U docker-compose；必要的时候安装wget或者pip

  2. docker－compose file & nginx 配置参考https://github.com/soft0626/docker-compose-nginx-ghost

  3. 其他：
    基本拉起来就能用，修改./data/ghost/config.js 中的网址设置。这份docker image默认ghost跑在开发状态下，目前看起来没什么大问题。摘要的显示也可以修改一下，参考https://yangsail.com/ghost-bo-ke-zhai-yao-mei-hua/
